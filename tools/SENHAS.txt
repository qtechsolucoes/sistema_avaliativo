═══════════════════════════════════════════════════════════
  🔧 SE ESQUECEU A SENHA
═══════════════════════════════════════════════════════════

OPÇÃO 1: Resetar Lockout (se bloqueado)

   1. Abra Console do navegador (F12)
   2. Digite: resetAuthLockout()
   3. Pressione Enter
   4. Tente novamente com senha correta

OPÇÃO 2: Ver Hash Atual (debug)

   1. Abra Console do navegador (F12)
   2. Digite: debugPasswordHash('admin123')
   3. Veja se o hash corresponde ao configurado

OPÇÃO 3: Mudar a Senha

   1. Abra: gerar-hash-senha.html no navegador
   2. Digite a nova senha desejada
   3. Clique em "Gerar Hashes"
   4. Copie os hashes gerados
   5. Cole no arquivo: src/utils/auth.js (linhas 18-19)

═══════════════════════════════════════════════════════════
  ⚙️ PARA MUDAR AS SENHAS
═══════════════════════════════════════════════════════════

1. Editar arquivo: src/env.js

   Linha 10: ADMIN_PASSWORD: 'sua_nova_senha',
   Linha 11: UNLOCK_PASSWORD: 'sua_nova_senha_unlock',

2. Gerar novos hashes:

   • Abra: gerar-hash-senha.html
   • Digite as novas senhas
   • Copie os hashes gerados

3. Atualizar arquivo: src/utils/auth.js

   Linha 18: this.adminPasswordHash = 'novo_hash';
   Linha 19: this.unlockPasswordHash = 'novo_hash';

4. Pronto! Senhas atualizadas.
