📋 Instruções para Teste

  1. Clique em "Ativar Modo Mock" para forçar o uso de dados mock
  2. Recarregue a página principal (index.html)
  3. Selecione ano "6", depois qualquer turma

  5. Selecione um estudante com adaptação para ver a interface personalizada
  6. Clique em "Desativar Modo Mock" quando terminar o teste
