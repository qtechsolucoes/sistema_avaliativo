═══════════════════════════════════════════════════════════════
  🔴 PROBLEMA: CHROMEBOOKS NÃO RODAM SERVIDOR LOCAL
═══════════════════════════════════════════════════════════════

Chromebooks têm sistema operacional bloqueado.
NÃO podem rodar Python, Node.js ou scripts locais.

═══════════════════════════════════════════════════════════════
  ✅ SOLUÇÃO: HOSPEDAR ONLINE (Obrigatório)
═══════════════════════════════════════════════════════════════

Você DEVE colocar o sistema na internet para funcionar em Chromebooks.

═══════════════════════════════════════════════════════════════
  🚀 MÉTODO 1: VERCEL (MAIS FÁCIL - 1 COMANDO)
═══════════════════════════════════════════════════════════════

1. Duplo clique em: DEPLOY_RAPIDO.bat

2. Siga as instruções na tela

3. Pronto! Link gerado: https://seu-projeto.vercel.app

4. Configure variáveis:
   - Acesse vercel.com
   - Projeto > Settings > Environment Variables
   - Adicione:
     * SUPABASE_URL = https://seu-projeto.supabase.co
     * SUPABASE_ANON_KEY = sua-chave-aqui

5. Compartilhe o link com os Chromebooks!

═══════════════════════════════════════════════════════════════
  🌐 MÉTODO 2: NETLIFY (SEM CÓDIGO - ARRASTE E SOLTE)
═══════════════════════════════════════════════════════════════

1. Selecione TODOS os arquivos desta pasta

2. Crie um arquivo ZIP

3. Acesse: https://app.netlify.com/drop

4. Arraste o ZIP para a página

5. Aguarde 1 minuto

6. Pronto! Link gerado: https://seu-site.netlify.app

7. Configure variáveis:
   - Site Settings > Environment variables
   - Adicione SUPABASE_URL e SUPABASE_ANON_KEY

8. Compartilhe o link!

═══════════════════════════════════════════════════════════════
  📱 MÉTODO 3: GITHUB PAGES (GRÁTIS)
═══════════════════════════════════════════════════════════════

1. Edite src/config.js (linha 20):

   const envVars = {
       'SUPABASE_URL': 'https://seu-projeto.supabase.co',
       'SUPABASE_ANON_KEY': 'sua-chave-completa-aqui'
   };

2. Suba para GitHub (use GitHub Desktop se não souber Git)

3. No repositório GitHub:
   - Settings > Pages
   - Source: main branch
   - Save

4. Link gerado: https://seu-usuario.github.io/sistema-avaliacoes

5. Compartilhe o link!

═══════════════════════════════════════════════════════════════
  🏫 MÉTODO 4: SERVIDOR DA ESCOLA (Rede Local)
═══════════════════════════════════════════════════════════════

⚠️ PRECISA: 1 computador que NÃO seja Chromebook

1. No computador do professor (Windows/Mac/Linux):

   python -m http.server 8000

2. Descubra o IP:

   Windows: ipconfig (procure IPv4)
   Mac/Linux: ifconfig

3. Nos Chromebooks, acesse:

   http://192.168.1.100:8000
   (substitua pelo IP real)

⚠️ Computador deve ficar ligado durante as provas!
